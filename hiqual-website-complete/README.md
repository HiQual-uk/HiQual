# HiQual United Kingdom Institute — Website

Official website for HiQual United Kingdom Institute (HSE Training & Certification).

## 📁 Folder Structure

```
hiqual-website/
│
├── index.html          → Homepage
├── style.css            → All website styling (colors, layout, responsive design)
├── script.js             → Mobile menu, form handling, small interactions
│
└── assets/
    └── images/
        ├── logo.png       → Institute logo (ADD THIS FILE)
        ├── about.jpg       → About section photo (ADD THIS FILE)
        ├── gallery1.jpg     → Gallery photos (ADD THESE FILES)
        ├── gallery2.jpg
        ├── gallery3.jpg
        ├── gallery4.jpg
        ├── gallery5.jpg
        └── gallery6.jpg
```

## ⚠️ Important: Images Needed

The homepage is coded to look for these image files inside `assets/images/`:
- `logo.png` — your institute logo
- `about.jpg` — a photo for the About Us section
- `gallery1.jpg` to `gallery6.jpg` — 6 photos for the Gallery section

**If these files are missing, the website will still work perfectly** — it will just show a navy/orange striped placeholder box instead of a blank broken image. You can add real photos any time later.
